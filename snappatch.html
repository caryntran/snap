Snap Patch

About

Snap! runs on MorphicJS which uses Adobe Flash Player. Because Adobe Flash Player is officially deprecated and is going to be unusable as of May 30th, we have designed a workaround. Snap Patch is essentially an open source version of Adobe Flash Player designed to run Snap! and be compatible with most modern web browsers.

© 2021 by Jens Mönig and Brian Harvey.

Downloads

Snap Patch is currently only supported on Unix and OpenBSD x86 architectures. More versions will be released soon.

MacOS x86 Installer Direct Download

Note

Because Snap Patch is open source and the code is publicly available, we are unable to sign it. This means that MacOS High Sierra+ does not allow you to run the installer. You can solve this is System Preferences > Security & Privacy.
